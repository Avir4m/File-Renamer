		  [--------------]
	   -==| File Renamer |==-
		  [--------------]


            Description
		     ---===---

Tools to rename all files in a folder by numbers.
example:
	file_1.png -> 0.png
	file_2.png -> 1.png

	ect..


			Instruction
		     ---===---
Put all the files you want to rename in the Files folder
Run the main.py
all files renamed to Renamed_Files
